# Dreamer Dash Forge

A Telegram Mini App for the Dreamers community — earn DR coins, participate in activities, join hackathons, and redeem rewards.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm run lint` — Run ESLint
- `npm run preview` — Preview production build
